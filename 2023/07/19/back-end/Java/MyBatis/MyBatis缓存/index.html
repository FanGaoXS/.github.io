MyBatis缓存简单梳理

1、什么是缓存（Cache）

存在内存里的临时数据

2、为什么要使用缓存

将一次查询的数据暂时放到缓存里，下次再次查询的时候就不用再查询数据库而是直接

查询缓存，这样提高了效率。减少与数据库交互次数。

3、什么样的数据可以使用缓存

经常查询，但是不经常修改的数据。

4、Mybatis缓存

Mybatis有一级缓存和二级缓存
